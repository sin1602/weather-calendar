☀️ 기상청 날씨 달력 자동화 제작 가이드

원본 저장소: github.com/Murianwind/weather-calendar

1 프로젝트 내 계정으로 가져오기 (Fork)

위 원본 저장소 링크로 접속한 뒤, 오른쪽 상단의 [Fork] 버튼을 눌러 본인의 GitHub 계정으로 복사하세요.

2 기상청 API 인증키 발급

1. 기상청 API 허브에 접속하여 회원가입을 진행합니다.

2. 로그인 후 상단 메뉴의 [마이페이지] → [인증키 현황]에서 나만의 인증키를 복사해두세요.

3 GitHub에 인증키 저장하기

1. 내 계정으로 가져온 저장소의 [Settings] → [Secrets and variables] → [Actions]로 이동합니다.

2. [New repository secret] 버튼을 클릭합니다.

3. Name에 KMA_API_KEY를 입력하고, Value에 복사한 인증키를 붙여넣어 저장합니다.

4 기상청 API 활용신청

기상청 사이트 왼쪽 메뉴에서 다음 항목들을 찾아 각각 [API활용신청] 버튼을 누르세요.

🏠 단기예보: 예특보 → 단기예보 → 4.3 단기예보조회
🌡️ 중기예보: 예특보 → 중기예보 → 2.2 중기기온조회 및 2.3 중기육상예보조회

신청 후 [마이페이지] → [활용신청 현황]에서 상태가 승인으로 표시되는지 확인하세요.

5 내 동네 설정값 찾기

동네 이름을 입력하면 코드에 넣을 좌표값을 알려드립니다.

👇 다음 단계에서 입력할 값들입니다:

  • NX: 60 / NY: 127
  • REG_ID_TEMP: 11B10101
  • REG_ID_LAND: 11B00000

6 내 동네 정보 입력 및 자동 생성

1. 내 저장소의 update_calendar.py 파일을 열고 연필 아이콘(Edit)을 누릅니다.

2. 5단계에서 얻은 좌표값으로 수정하고 하단의 [Commit changes...]를 눌러 저장합니다.

⏳ 잠시만 기다려주세요!
저장 버튼을 누르면 GitHub이 자동으로 날씨 파일을 만들기 시작합니다. 상단 [Actions] 탭에서 작업이 완료(초록색 체크)될 때까지 약 1~2분 정도 기다려주세요. 작업이 끝나야 weather.ics` 파일이 생겨납니다.

7 캘린더 주소 복사 및 구독

1. 빌드가 완료되었다면 저장소 메인 화면에 생긴 weather.ics 파일을 클릭합니다.

2. 우측 상단의 [Raw] 버튼을 누르면 나타나는 텍스트 화면의 브라우저 주소를 복사합니다.

3. 아이폰/갤럭시 달력 앱의 '구독 캘린더' 메뉴에 해당 주소를 붙여넣으면 모든 설정이 완료됩니다!

🎉 **설정이 완료되면 아래와 같이 달력 앱에서 매일의 날씨와 시간별 예보를 확인할 수 있습니다.**

(캘린더 예시 화면)
